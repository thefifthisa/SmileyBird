I'm a big fan of romantic comedies and coordinating outfits, so my overarching idea is to create a wardrobe app inspired by Cher's closet computer in the movie Clueless.

I envision that the user can mix and match pieces from different categories in the curated wardrobe or choose "auto-dress" mode for a randomly generated outfit. I think it would be another interesting challenge to implement the original closet computer's "mismatch" feature which flags outfits that aren't as well-coordinated, possibly by encoding which colors and styles match.
