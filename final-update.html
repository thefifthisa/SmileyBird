Final Project Update #1

After settling on my new idea of a smile-controlled Flappy Bird, I got to work right away with setting up the game environment. I used the Face Tracking sample code to start a video capture using the player's webcam and track the different points on their lips using the CLM tracker. I then searched for pixel art assets for Flappy, the city background, and the other icons needed while playing. Before placing them in the game, I edited some of the colors using Pixel Art Maker to have a cohesive style to the game.

The game utilizes object-oriented design with classes for Flappy and Pipes. Implementing the first, I struggled with how to control Flappy using the player's smile. First, I printed the player's mouth opening, calculated using points on the CLM tracker, to the console in order to determine its minimum and maximum values. I then used the map function to translate that into a speed that gets added to Flappy's y-position and makes it move up or down accordingly. However, I found that the resulting movement was a little too slow compared to the original Flappy Bird. I realized a better approach would be to map the mouth opening to a y-position based on the height of the canvas, but it's currently a little too jittery.

In terms of next steps, I'll tinker more with the position mapping and also get started on implementing the moving pipes that Flappy needs to go through.

Final Project Update #2

Everything's coming together! Since my last update, I implemented the Pipes class so the game is fully playable now.

One of the biggest challenges I faced in this part is figuring out when to add a new pipe. There was a point when my game always crashed upon starting because I was inadvertently adding so many pipes at once. I later realized the problem and added checks to only add a new pipe if Flappy either passed or hit the current one. Flappy's movement is also less jittery now after I changed the map function to a smaller range of y-positions, but I'll continue tweaking to get it just right.

I also created title and end screens with animations!
